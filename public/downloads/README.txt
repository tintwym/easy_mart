Place your signed release APK here as:
  easymart.apk

Then the /download page will offer it for download.

To build the APK:
  1. Open Android Studio: npx cap open android
  2. Build → Build Bundle(s) / APK(s) → Build APK(s)
     (Or Build → Generate Signed Bundle / APK → APK)
  3. Copy the built APK (e.g. app/build/outputs/apk/release/app-release.apk)
     to this folder and rename it to easymart.apk

See docs/MOBILE_APP.md for full instructions.
